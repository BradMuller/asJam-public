# asJam, a jam of ActionScript

asJam converts ActionScript 3 source code into [Spaceport][1]-compatible
JavaScript source code.

asJam is copyright (c) 2011-2012 spaceport.io, Inc.; see LICENSE for details.

[1]: http://spaceport.io/

## Setting up

Binary dependencies:

* Node.JS 0.4.x, 0.6.x

To install:

    npm install

To test:

    npm test
    # or
    make test
